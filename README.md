# egg-wechat

> Egg plugin for wechat


## 安装

```bash
$ npm i egg-wechat2 --save
# or
$ yarn add egg-wechat2
```

## 依赖说明

### 依赖的 egg 版本

egg 2.x | egg 1.x
--- | ---
😁 | ❌

### 依赖的插件

[tnwx](https://github.com/javen205/TNWX)

## 开启插件

```js
// {app_root}/config/plugin.js
exports.wechat = {
  enable: true,
  package: 'egg-wechat',
};
```

## 配置

```js
// {app_root}/config/config.default.js
config.wechat = {

};
```
